# rm Command

## What is rm?

The `rm` command is used to remove files and directories.

## Syntax

```bash
rm [options] file_name
```

## Delete a File

```bash
rm notes.txt
```

## Delete Multiple Files

```bash
rm file1.txt file2.txt
```

## Delete a Directory Recursively

```bash
rm -r myfolder
```

## Force Delete

```bash
rm -rf myfolder
```

## Warning

Files deleted with `rm` cannot be easily recovered.
