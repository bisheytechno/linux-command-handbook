# cat Command

## What is cat?

The `cat` command is used to display, create, and combine text files.

## Syntax

```bash
cat filename
```

## Example

```bash
cat notes.txt
```

Displays the contents of `notes.txt`.

## Display Multiple Files

```bash
cat file1.txt file2.txt
```

Displays the contents of both files.

## Create a File

```bash
cat > hello.txt
```

Type content and press `Ctrl + D` to save.

## Real World Example

```bash
cat /etc/passwd
```

Displays user account information stored in the `/etc/passwd` file.

## Sample Output

```bash
Hello World
Welcome to Linux
```
