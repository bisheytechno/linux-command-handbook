# echo Command

## What is echo?

The `echo` command is used to display text or variables in the terminal.

## Syntax

```bash
echo [text]
```

## Example

```bash
echo Hello World
```

## Output

```bash
Hello World
```

## Display Environment Variable

```bash
echo $HOME
```

Displays the current user's home directory.

## Write Text to a File

```bash
echo "Welcome to Linux" > message.txt
```

Creates a file and writes text to it.

## Real World Example

```bash
echo "Backup completed successfully"
```

Displays a status message in the terminal.
