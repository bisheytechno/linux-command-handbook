# diff Command

## What is diff?

The `diff` command compares two files line by line.

## Syntax

```bash
diff file1 file2
```

## Example

```bash
diff old.txt new.txt
```

## Side-by-Side Comparison

```bash
diff -y old.txt new.txt
```

## Real World Example

```bash
diff config-old.conf config-new.conf
```
