# vmstat Command

## What is vmstat?

The `vmstat` command displays information about memory, processes, CPU activity, and system performance.

## Syntax

```bash
vmstat
```

## Example

```bash
vmstat
```

## Update Every 2 Seconds

```bash
vmstat 2
```

## Real World Example

```bash
vmstat 2 5
```

Displays system statistics every 2 seconds for 5 updates.
