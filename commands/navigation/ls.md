# ls Command

## What is ls?

The `ls` command is used to list files and directories.

## Syntax

```bash
ls [options] [directory]
```

## Basic Example

```bash
ls
```

## Common Options

| Option | Description |
|---------|------------|
| -a | Show hidden files |
| -l | Long format |
| -h | Human readable file sizes |

## Real World Example

```bash
ls -lah
```

## Sample Output
