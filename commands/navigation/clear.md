# clear Command

## What is clear?

The `clear` command is used to clear the terminal screen.

## Syntax

```bash
clear
```

## Example

```bash
clear
```

Clears all previous commands and output from the terminal window.

## Real World Example

```bash
ls
pwd
clear
```

After running `clear`, the terminal screen becomes clean while the command history remains available.

## Alternative Shortcut

```bash
Ctrl + L
```

This keyboard shortcut performs the same action as the `clear` command.
